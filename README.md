# 🤖 AI Tutor Project (with RAG)

## 📌 Description
This project is an AI-based tutor that helps users learn concepts using **Retrieval-Augmented Generation (RAG)**.  
It retrieves relevant information from documents and generates accurate, context-aware responses.

---

## 🚀 Features
- 🤖 AI-powered intelligent responses  
- 📄 Document-based learning (RAG)  
- ⚡ Fast and accurate answers  
- 🔍 Context-aware question answering  

---

## 🛠️ Technologies Used
- Python  
- OpenAI API  
- Machine Learning  
- RAG (Retrieval-Augmented Generation)

---

## ⚙️ Setup Instructions

### 1️⃣ Clone the repository

git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name

### 2️⃣ Install dependencies
pip install -r requirements.txt

### 3️⃣ Add API Key (IMPORTANT)
Create a .env file:
Environment
OPENAI_API_KEY=your_api_key_here

### 4️⃣ Run the project
python app.py
